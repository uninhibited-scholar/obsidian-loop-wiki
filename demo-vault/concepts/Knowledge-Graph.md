# Knowledge Graph

A set of typed notes connected by links, so that knowledge forms a *network*
rather than a pile of isolated pages. In this vault the connections are
[[Wikilink]]s, and the graph is rendered by [[Obsidian]]'s graph view.

A node with zero links is an orphan — a silent failure. The whole point of
[[Loop-Engineering]] applied to a vault is that the verification pass can check,
mechanically, that the graph is actually connected.
(source: raw/articles/loop-engineering-source.md)
