# Wikilink

A link between notes written with double square brackets — for example a
reference to [[Obsidian]]. The single most important convention in this vault:
without wikilinks, pages stay islands and the [[Knowledge-Graph]] never forms —
the Obsidian graph view shows nothing.

Rule: every new node links to at least one existing node, and important concepts
are always written as wikilinks. This is the foundational prerequisite the
verification pass in [[Loop-Engineering]] enforces.
