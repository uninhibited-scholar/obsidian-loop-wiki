# Claude Code

Anthropic's agentic CLI. Relevant here because it productizes the micro version
of [[Loop-Engineering]]: a `/goal` command runs round after round until a
completion condition is met, and `/loop` repeats a task on an interval.

This vault's `loop-wiki` skill installs into Claude Code, but the same workflow
runs on any agent via `AGENTS.md`.
