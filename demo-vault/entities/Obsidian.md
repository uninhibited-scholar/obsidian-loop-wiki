# Obsidian

A local-first Markdown note app. Two features make it the storage layer for this
project: the **graph view** (visualizes the [[Knowledge-Graph]]) and
**backlinks** driven by [[Wikilink]]s.

Open a vault with *Open folder as vault*, pointing at the whole vault root — not
a subfolder — so the agent and Obsidian share the same files.

See also: [[Loop-Wiki-Map]]
